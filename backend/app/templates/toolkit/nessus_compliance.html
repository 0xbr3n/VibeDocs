What's in the workbook

  • Summary — total rows, unique hosts / policies, result breakdown (FAILED / PASSED / WARNING / …), benchmark-level breakdown.
  • All Compliance — every parsed row with full detail: host, IP, FQDN, OS, policy, plugin ID, benchmark level + profile, policy setting, requirement description, solution, result, actual / policy values, severity, service / port / protocol, see-also, synopsis, description.
  • Host Summary — one row per host with result counts.
  • One sheet per policy — easy to hand a specific benchmark sheet to a system owner.

Cells whose value starts with = are prefixed with a single quote so Excel doesn't treat them as formulas. Output otherwise matches the standalone nessus_compliance_to_excel.py CLI byte-for-byte.
