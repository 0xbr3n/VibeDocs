What's in the workbook

  • CIS Mapping — one row per (client control, CIS benchmark): Custom Control, CIS Benchmark title, CIS Control ID(s), and the source snippet it was extracted from.
  • Benchmarks — the distinct "CIS … Benchmark vX.Y.Z" titles the document referenced.
  • Summary — extraction counts + any warnings (e.g. CIS ids that couldn't be tied to a client control).

The extractor recognises dotted CIS control ids (e.g. 1.2.3, 18.9.45.1), prefixed ids (CIS 1.2.3, Recommendation 5.2), and client ids like HCR-001 / STD-1.4 / SEC.3.2. Cells beginning with = are escaped so Excel never treats extracted text as a formula.
