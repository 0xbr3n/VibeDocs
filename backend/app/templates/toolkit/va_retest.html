What's in the ZIP

  • updated_tracker.xlsx — your tracker, in-place edits applied. Embedded images (screenshots, badges, etc.) are preserved.
  • new_hosts.xlsx — every rescan row landing on a net-new IP (only when the new-IP action is "include" or "list_only").
  • summary.txt — counts + the IPs the pipeline flagged as net-new, including affected row counts per IP.

Rows whose Status is already Closed (or a synonym like Remediated / Fixed / Resolved) are left untouched — the script is safely re-runnable against the same tracker without re-processing rows you've already decided on.
